# 📊 Sales Dashboard using Power BI
Interactive Sales Dashboard created using Power BI for sales trend and regional analysis.

## 🚀 Project Overview
This project presents an interactive **Sales Dashboard** built using **Power BI** to analyze sales performance and identify business trends. The dashboard provides insights into category-wise sales, regional performance, and overall sales distribution through interactive visualizations.

---

## 🛠️ Tools & Technologies Used
- Power BI  
- CSV Dataset  
- Data Visualization  
- Dashboard Design  

---

## 📈 Dashboard Features
✅ Total Sales Overview  
✅ Category-wise Sales Analysis  
✅ Region-wise Sales Distribution  
✅ Sales Trend Over Time  
✅ Sub-Category Performance Analysis  
✅ Interactive Region Filter (Slicer)  

---

## 📊 Key Insights
- Identified top-performing product categories  
- Compared sales distribution across regions  
- Analyzed sales trends using visual reports  
- Created an interactive dashboard for easy exploration of data
